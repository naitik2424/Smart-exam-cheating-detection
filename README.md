# 🚀 Smart Exam Cheating Detection System

## 📌 Overview
This is a DBMS project designed to detect cheating during exams by tracking student activities.  
The system uses a MariaDB database and SQL queries to identify suspicious behavior.

---

## 🛠️ Tech Stack
- Database: MariaDB
- Language: SQL
- Tools: XAMPP / VS Code / MySQL Workbench

---

## 🧱 Database Design
The database includes:
- Students  
- Exams  
- Activity Logs  
- Cheating Records  

The database is normalized up to 3NF to remove redundancy and ensure consistency.
